HTML- jest to szkielet strony.
< body > - ciało strony.
< main > - główna część dokumentu.
< section > - Sekcja dokumentu (rozdział).
< article > - Samodzielna treść (podrozdział).
< header > - Nagłówek dokumentu / sekcji / strony.
< footer> - Stopka sekcji / artykułu.
< h1-h6 > - Nagłówki.
< p > - Paragraf / akapit.

< br / > - brake line, koniec linijki tekstu.
< b > - bold
< i > - italc
< strong > - somatyczny bold.
< em > - somatyczny italic.
< a> - link / łącze w dokumencie.
